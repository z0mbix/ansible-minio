# Ansible Minio

## Description

Install and start a single node Minio server

Tested on CentOS 7 on the following architectures:

*  i386
*  x86_64
*  aarch64
*  armv7l

Mostly tested on [Scaleway](https://www.scaleway.com/) and vagrant with test-kitchen.

## Testing

Test using test-kitchen:

```
$ kitchen test
```
